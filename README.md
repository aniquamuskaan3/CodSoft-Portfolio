# 🚀 Modern Portfolio Website

A modern, responsive, and visually stunning personal portfolio website built using HTML, CSS, and JavaScript.

This portfolio showcases skills, projects, personal information, and contact details through a clean user interface, smooth animations, and a professional design.

## ✨ Features

- Modern Glassmorphism UI
- Fully Responsive Design
- Animated Hero Section
- About Me Section
- Skills Showcase
- Project Gallery
- Contact Information Section
- Smooth Scroll Effects
- Scroll Reveal Animations
- Social Media Integration
- Mobile-Friendly Layout

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 🚀 Getting Started

1. Download or Clone the Repository

```bash
git clone https://github.com/your-username/ultimate-portfolio.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
